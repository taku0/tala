Turtle: Terse RDF Triple Language

RDFはグラフ構造を表すデータ構造。(元ノード, エッジ, 先ノード)というタプルの集合。
RDF is a data structure for graph structure. Set of tuple (source, edge, destination)

  • コンパクト / Compact
  • 型を指定できる / Typed (optional)
  • 循環構造を表現できる / Cyclic data structures

コンパクト / Compact

Map:

JSON:
{
  "name": "foo",
  "age": 20
}
Turtle:
[
  :name "foo";
  :age 20
].

Array

JSON:
[1, 2, 3]
Turtle:
(1, 2, 3)

Primitives

JSON:
  • 123.456
  • "abc"
  • true, false
  • null
Turtle:
  • 123.456
  • "abc"
  • """multi
    line
    string"""
  • true, false

型を指定できる / Types (optional)

a Person named "foo", 20 years old:
[
  a :Person;
  :name "Foo";
  :age 20
]
  • "-5" ^^ xsd:integer
  • "-5" ^^ xsd:double
  • "-5" ^^ your:customType

循環構造を表現できる / Cyclic data structure

Scala
val ones: Stream[Int] = 1 #:: ones
// Stream(1, ...)
Turtle
_:ones rdf:first 1;
       rdf:rest _:ones.
